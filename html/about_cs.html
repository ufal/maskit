MasKIT je webový nástroj a služba REST API pro pseudonymizaci a anonymizaci českých právních textů. Vstupní text (např. dopis zaslaný právním orgánem občanovi) je zpracován pomocí externích služeb pro větný rozbor a rozpoznání jmenných entit. Program poté v textu identifikuje a nahradí osobní informace.
